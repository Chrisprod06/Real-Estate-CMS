Εγχειρίδιο Χρήσης Διαχείριση Ακινήτων

Σε αυτή την οθόνη μπορείτε να δείτε πληροφορίες για τα ακίνητα που υπάρχουν στο συστήμα, ναπροσθέσετε λεπτομέρειες φωτογραφίες και βίντεο για νέα ακίνητα καθώς και να τα διαγράψετε ή να επεξεργαστείτε τις λεπτομέρειες κάποιου ακινήτου. Επίσης μπορείτε να δημιουργήσετε αναφορά για τα ακίνητα τις επιλογής σας.

  1. Εισαγωγή λεπτομερειών νέου ακινήτου:

    Αρχικά πατήστε το κουμπί Προσθήκη Νέου Ακινήτου και θα ανοίξει ένα παράθυρο . Στη συνέχεια εισάγετε όλα τα στοιχεία ( * σημαίνει ότι είναι υποχρεωτικό ) και πατήστε το κουμπί Προσθήκη.

  2. Εισαγωγή φωτογραφιών και βίντεο νέου ακινήτου:

    Αρχικά πατήστε το κουμπί Προσθήκη Νέου Ακινήτου. Στη συνέχεια διαλέξετε για ποιό ακίνητο θέλετε να εισάγετε φωτογραφίες και βίντεο ( * σημαίνει ότι είναι υποχρεωτικό ). Μπορείτε να εισάγετε φωτογραφίες για να εμφανίζονται στη προεπισκόπηση, στα καρουσέλ, αλλα και στο panorama view. Η φωτογραφία για τη προεπισκόπηση και για το καρουσέλ της αρχικής σελίδας πρέπει ΠΑΝΤΑ να είναι πρώτη. Μέτα μπορείτε να εισάγετε τις υπόλοιπες φωτογραφίες με όποια σειρά επιθυμήτε. και πατήστε το κουμπί Προσθήκη.

  3. Επεξεργασία λεπτομερειών υπάρχων ακινήτου:

    Αρχικά πατήστε το εικονίδιο του μολυβιού το οποίο βρίσκεται στο τελευταία στήλη στο πίνακα ακινήτων και θα ανοίξει ένα παράθυρο με όλα τα υπάρχων στοιχεία για το συγκεκριμένο ακίνητο. Όσα στοιχεία στοιχεία υποστούν αλλαγές θα ανανεωθούν στο σύστημα ενώ τα υπόλοιπα θα κρατήσουν της αρχικές τους τιμές.

  4. Επεξεργασία φωτογραφιών και βίντεο ακινήτου:

  5. Διαγραφή λεπτομερειών, φωτογραφιών και βίντεο ακινήτου:

    Αρχικά πατήστε το εικονίδιο του καλάθου το οποίο βρίσκεται στο τελευταία στήλη στο πίνακα ακινήτων και θα ανοίξει ένα παράθυρο το οποίο σας ρωτά αν θέλετε να διαγράψετε το ακίνητο. Μετά πατήστε διαγραφή και θα διαγραφτούν όλα τα στοιχεία του ακινήτου καθώς και ότι φωτογραφίες και βίντεο υπάρχουν στο σύστημα για το συγκεκριμένο ακίνητο.

  6. Δημιουργία αναφοράς:

    Αρχικά πατήστε το κουμπί Δημιουργία Αναφοράς και θα εμφανιστεί ένα παράθυρο. Σε αυτο το παραθυρό επιλέξετε τις προτιμήσεις σας σχετικά με το για ποιά ακίνητα θέλετε να δημιουργήσετε αναφορά. Σε περιπτωσή που δεν έχετε κάποια προτιμήση μπορείτε να μην επιλέξετε κάτι. Τέλος πατήστε το κουμπί Δημιουργία.

  7. Περιγραφές Πεδίων:

    • Τύπος: Εδώ μπορείτε να διαλέξετε αν το ακίνητο είναι σπίτι ή διαμέρισμα.

    • Κατηγορία: Εδώ μπορείτε να διαλέξετε αν το ακίνητο είναι για πούλημα, ενοικίαση βραχυπρόδεσμα ή μακροπρόθεσμα, ανακαίνιση ή διακόσμηση.

    • Χώρα: Εδώ μπορείτε να πληκτρολογίσετε σε ποιά χώρα βρίσκεται το ακίνητο.

    • Πόλη: Εδώ μπορείτε να πληκτρολογίσετε σε ποιά πόλη βρίσκεται το ακίνητο.

    • Περιφέρεια: Εδώ μπορείτε να πληκτρολογίσετε σε ποιά περιφέρεια βρίσκεται το ακίνητο.

    • Διεύθυνση: Εδώ μπορείτε να πληκτρολογίσετε τη διεύθυνση του ακινητού.

    • Υπνοδωμάτια: Εδώ μπορείτε να διαλέξετε πόσα υπνοδωμάτια έχει το ακίνητο.

    • Αποχωρητήρια: Εδώ μπορείτε να διαλέξετε πόσα αποχωρητήρια έχει το ακίνητο.

    • Επιπλα: Εδώ μπορείτε να διαλέξετε αν το ακίνητο έχει επιπλά ή όχι.

    • Χώρος Σταθμευσής: Εδώ μπορείτε να διαλέξετε αν το ακίνητο έχει χώρος στάθμευσής ή όχι.

    • Συστημα Θέρμανσης: Εδώ μπορείτε να διαλέξετε αν το ακίνητο έχει σύστημα Θέρμανσης ή όχι.

    • Όροφος: Εδώ μπορείτε να διαλέξετε τον όροφο που βρίσκεται αν είναι διαμέρισμα ή τους ορόφους που έχει αν είναι σπίτι το ακίνητο.

    • Χρόνος Κατασκευής: Εδώ μπορείτε να πληκτρολογήσετε τη χρονολογία κατασκευής του ακινητού. Πρέπει να είναι το μέγιστο 4 ψηφία.

    • Διαθεσιμό Από: Εδώ μπορείτε να διαλέξετε την ημερομηνία που θα είναι διαθέσιμο το ακίνητο.

    • Τετραγωνικά Μέτρα: Εδώ μπορείτε να πληκτρολογήσετε πόσα τετραγωνικά είναι το ακίνητα.

    • Συνολική Τιμή: Εδώ μπορείτε να πληκτρολογήσετε τη συνολική τιμή του ακινήτου.

    • Εμφάνιση στην αρχική: Εδώ μπορείτε να επιλέξετε αν θέλετε το ακίνητο να εμφανίζεται στο καρουσέλ της αρχικής σελίδας.

    • Τοποθεσία: Εδώ μπορείτε να εισάγετε το χάρτη της τοποθεσίας του ακινήτου. Για να δουλέψει σωστά αυτή η λειτουργεία ΠΡΕΠΕΙ πρώτα να πάτε στη σελίδα https://www.google.com/maps/ . Έπειτα να διαλέξετε τη τοποθεσία του ακινήτου. Μετά πρέπει να πατήσετε στη όδο που εμφανίζει στο παραθυράκι στο κάτω μέρος της σελίδας. Μετά στη καρτέλα που θα εμφανιστεί πρέπει να πατήσετε το κουμπί share ή κοινοποίηση και μετά το embed a map. Τέλος βεβαιωθείτε πώς είναι επιλεγμένο το μέγεθος medium και πατήστε COPY HTML και κάντε το paste ή επικόλληση σε αυτό το πεδίο.

    • Περιγραφή: Εδώ μπορείτε να πληκτρολογήσετε τη περιγραφή του ακινήτου. Το μέγιστο μέγεθος πρέπει να είναι 1200 χαρακτήρες.

    • Ανέσεις: Εδώ μπορείτε να πληκτρολογήσετε τις ανέσεις που έχει το ακίνητο. Το μέγιστο μέγεθος πρέπει να είναι 1200 χαρακτήρες.

    • Φωτογραφίες: Πατώντας αυτό το το κουμπί μπορείτε να ανεβάσετε φωτογραφίες για ένα συγκεκριμένο ακίνητο προκειμένου να μπορούν να προβληθούν στο καρουσέλ του αντίστοιχου ακινήτου στην ιστοσελίδα, αφού έχτε πρώτα επιλέξει τον αριθμό του ακινήτου για τον οποίο επιθυμήτε να ανεβάσετε αρχεία. Όταν πατήσετε το κουμπί choose files, μπορείτε να επιλέξετε το πολύ δέκα(10) φωτογραφίες του τύπου .png και .jpeg έχωντας πατημένο το κουμπί ctrl, ούτως ώστε να ανεβούν όλες οι φωτογραφίες ταυτόχρονα. Να σημειωθεί επίσης ότι το μέγιστο μέγεθος για κάθε αρχείο πρέπει να είναι το πολύ 20ΜΒ. Η φωτογραφία πού θα ανεβεί πρώτη θα είναι καί η φωτογραφία που θα εμφανίζεται στο thumbnail του property grid στην ιστοσελίδα. Επίσης οι φωτογραφίες πρέπει να επιλεχθούν ανάποδα, ανάλογα με την σειρά που θέλετε να τις ανεβάσετε. Για παράδειγμα αν η σειρά που θέλετε να ανεβάσετε τα αρχεία είναι(photo1, photo2, photo3) η επιλογή απο τον υπολογιστή σας πρέπει να είναι(photo3, photo2, photo1).

    • Φωτογραφίες Πανόραμα: Πατώντας αυτό το το κουμπί μπορείτε να ανεβάσετε φωτογραφίες Πανόραμα για ένα συγκεκριμένο ακίνητο προκειμένου να μπορούν να προβληθούν στην γκαλερύ του αντίστοιχου ακινήτου στην ιστοσελίδα, αφού έχτε πρώτα επιλέξει τον αριθμό του ακινήτου για τον οποίο επιθυμήτε να ανεβάσετε αρχεία. Όταν πατήσετε το κουμπί choose files, μπορείτε να επιλέξετε το πολύ δέκα(10) αρχεία του τύπου .jpg έχωντας πατημένο το κουμπί ctrl, ούτως ώστε να ανεβούν όλες οι φωτογραφίες ταυτόχρονα. Να σημειωθεί επίσης ότι το μέγιστο μέγεθος για κάθε αρχείο πρέπει να είναι το πολύ 20ΜΒ. Επίσης οι φωτογραφίες πρέπει να επιλεχθούν ανάποδα, ανάλογα με την σειρά που θέλετε να τις ανεβάσετε. Για παράδειγμα αν η σειρά που θέλετε να ανεβάσετε τα αρχεία είναι(photo1, photo2, photo3) η επιλογή απο τον υπολογιστή σας πρέπει να είναι(photo3, photo2, photo1).

    • Βίντεο: Πατώντας αυτό το το κουμπί μπορείτε να ανεβάσετε το βίντεο για ένα συγκεκριμένο ακίνητο προκειμένου να μπορεί να προβληθεί στην ιστοσελίδα, αφού έχτε πρώτα επιλέξει τον αριθμό του ακινήτου για τον οποίο επιθυμήτε να ανεβάσετε αρχεία. Όταν πατήσετε το κουμπί choose files, μπορείτε να επιλέξετε το ένα(1) αρχείο του τύπου .mp4. Να σημειωθεί επίσης ότι το μέγιστο μέγεθος γιά το αρχείο πρέπει να είναι το πολύ 20ΜΒ.

    • Αντικατάσταση αρχείων: Σε περίπτωση που θέλετε να ανεβάσετε κανούργια αρχεία για το κάθε ακίνητο η διαδικασία είναι ακριβώς η ίδια με τα βήματα που αναφέρονται πιό πάνω. Να σημειωθεί ότι σε περιπτωσή αντικατάστασης των παλιών αρχείων με καινούργια, τα παλιά αρχεία θα διαγραφτούν απο το σύστημα. Αυτό σημαίνει ότι όταν θέλετε να αντικαταστήσετε για παράδειγμα μία μονο φωτογραφία, θα πρέπει να ξανα ανεβάσετε και τα παλιά σας αρχεία ξανά απο την αρχή.
