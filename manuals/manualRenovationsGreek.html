Εγχειρίδιο Χρήσης Διαχείριση Ανακαινίσεων

Σε αυτή την οθόνη μπορείτε να δείτε πληροφορίες για τις ανακαίνισεις που υπάρχουν στο συστήμα, ναπροσθέσετε λεπτομέρειες φωτογραφίες και βίντεο για νέες ανακαίνισεις καθώς και να τα διαγράψετε ή να επεξεργαστείτε τις λεπτομέρειες κάποιας ανακαίνισης. Επίσης μπορείτε να δημιουργήσετε αναφορά για τις ανακαίνισεις τις επιλογής σας.

  1. Εισαγωγή λεπτομερειών νέας ανακαίνισης:

    Αρχικά πατήστε το κουμπί Προσθήκη νέας ανακαίνισης και θα ανοίξει ένα παράθυρο . Στη συνέχεια εισάγετε όλα τα στοιχεία ( * σημαίνει ότι είναι υποχρεωτικό ) και πατήστε το κουμπί Προσθήκη.

  2. Εισαγωγή φωτογραφιών και βίντεο νέας ανακαίνισης:

    Αρχικά πατήστε το κουμπί Προσθήκη Νέας Ανακαίνισης. Στη συνέχεια διαλέξετε για ποιά ανακαίνιση θέλετε να εισάγετε φωτογραφίες και βίντεο ( * σημαίνει ότι είναι υποχρεωτικό ). Μπορείτε να εισάγετε φωτογραφίες για να εμφανίζονται στη προεπισκόπηση, στα καρουσέλ, αλλα και στο panorama view. Η φωτογραφία για τη προεπισκόπηση και για το καρουσέλ της αρχικής σελίδας πρέπει ΠΑΝΤΑ να είναι πρώτη. Μέτα μπορείτε να εισάγετε τις υπόλοιπες φωτογραφίες με όποια σειρά επιθυμήτε. και πατήστε το κουμπί Προσθήκη.

  3. Επεξεργασία λεπτομερειών υπάρχων ανακαίνισης:

    Αρχικά πατήστε το εικονίδιο του μολυβιού το οποίο βρίσκεται στο τελευταία στήλη στο πίνακα ανακαινίσεων και θα ανοίξει ένα παράθυρο με όλα τα υπάρχων στοιχεία για τη συγκεκριμένη ανακαίνιση. Όσα στοιχεία υποστούν αλλαγές θα ανανεωθούν στο σύστημα ενώ τα υπόλοιπα θα κρατήσουν της αρχικές τους τιμές.

  4. Επεξεργασία φωτογραφιών και βίντεο ανακαίνισης:

  5. Διαγραφή λεπτομερειών, φωτογραφιών και βίντεο ανακαίνισης:

    Αρχικά πατήστε το εικονίδιο του καλάθου το οποίο βρίσκεται στο τελευταία στήλη στο πίνακα ανακαινίσεων και θα ανοίξει ένα παράθυρο το οποίο σας ρωτά αν θέλετε να διαγράψετε την ανακαίνιση. Μετά πατήστε διαγραφή και θα διαγραφτούν όλα τα στοιχεία του ανακαίνισης καθώς και ότι φωτογραφίες και βίντεο υπάρχουν στο σύστημα για τη συγκεκριμένη ανακαίνιση.

  6. Δημιουργία αναφοράς:

    ταν πατήσετε το κουμπί Δημιουργία Αναφοράς θα δημιουργηθεί μια αναφορά με τα στοιχεία όλων των ανακαινίσεων σε μορφη pdf.

  7. Περιγραφές Πεδίων:

    • ID ακινήτου: Εδώ μπορείτε να διαλέξετε για ποιό ακίνητο θέλετε να προσθέσετε ανακαίνιση ή διακόσμηση. Πρέπει να είναι ακινητό με κατηγορία για ανακαίνιση ή διακόσμηση

    • Περιγραφή: Εδώ μπορείτε να πληκτρολογήσετε τη περιγραφή του ανακαίνισης. Το μέγιστο μέγεθος πρέπει να είναι 1200 χαρακτήρες.

    • Φωτογραφίες Πριν: Πατώντας αυτό το το κουμπί μπορείτε να ανεβάσετε φωτογραφίες πριν την διακόσμηση/ανακαίνιση του συγκεκριμένο ακίνητο προκειμένου να μπορούν να προβληθούν στο καρουσέλ του αντίστοιχου ακινήτου στην ιστοσελίδα, αφού έχτε πρώτα επιλέξει τον αριθμό του ακινήτου για τον οποίο επιθυμήτε να ανεβάσετε αρχεία. Όταν πατήσετε το κουμπί choose files, μπορείτε να επιλέξετε το πολύ τέσσερις(4) φωτογραφίες του τύπου .png, .jpeg και .jpg έχωντας πατημένο το κουμπί ctrl, ούτως ώστε να ανεβούν όλες οι φωτογραφίες ταυτόχρονα. Να σημειωθεί επίσης ότι το μέγιστο μέγεθος για κάθε αρχείο πρέπει να είναι το πολύ 20ΜΒ. Η φωτογραφία πού θα ανεβεί πρώτη θα είναι καί η φωτογραφία που θα εμφανίζεται στο thumbnail του renovation grid στην ιστοσελίδα. Επίσης οι φωτογραφίες πρέπει να επιλεχθούν ανάποδα, ανάλογα με την σειρά που θέλετε να τις ανεβάσετε. Για παράδειγμα αν η σειρά που θέλετε να ανεβάσετε τα αρχεία είναι(photo1, photo2, photo3) η επιλογή απο τον υπολογιστή σας πρέπει να είναι(photo3, photo2, photo1).

    • Φωτογραφίες Μετά: Πατώντας αυτό το το κουμπί μπορείτε να ανεβάσετε φωτογραφίες μετά την διακόσμηση/ανακαίνιση του συγκεκριμένο ακίνητο προκειμένου να μπορούν να προβληθούν στο καρουσέλ του αντίστοιχου ακινήτου στην ιστοσελίδα, αφού έχτε πρώτα επιλέξει τον αριθμό του ακινήτου για τον οποίο επιθυμήτε να ανεβάσετε αρχεία. Όταν πατήσετε το κουμπί choose files, μπορείτε να επιλέξετε το πολύ τέσσερις(4) φωτογραφίες του τύπου .png, .jpeg και .jpg έχωντας πατημένο το κουμπί ctrl, ούτως ώστε να ανεβούν όλες οι φωτογραφίες ταυτόχρονα. Να σημειωθεί επίσης ότι το μέγιστο μέγεθος για κάθε αρχείο πρέπει να είναι το πολύ 20ΜΒ. Η φωτογραφία πού θα ανεβεί πρώτη θα είναι καί η φωτογραφία που θα εμφανίζεται στο thumbnail του renovation grid στην ιστοσελίδα. Επίσης οι φωτογραφίες πρέπει να επιλεχθούν ανάποδα, ανάλογα με την σειρά που θέλετε να τις ανεβάσετε. Για παράδειγμα αν η σειρά που θέλετε να ανεβάσετε τα αρχεία είναι(photo1, photo2, photo3) η επιλογή απο τον υπολογιστή σας πρέπει να είναι(photo3, photo2, photo1).

    • Βίντεο Πριν: Πατώντας αυτό το το κουμπί μπορείτε να ανεβάσετε το βίντεο πριν την διακόσμηση για ένα συγκεκριμένο ακίνητο προκειμένου να μπορεί να προβληθεί στην ιστοσελίδα, αφού έχτε πρώτα επιλέξει τον αριθμό της διακόσμησης/ανακαίνισης για τον οποίο επιθυμήτε να ανεβάσετε αρχεία. Όταν πατήσετε το κουμπί choose files, μπορείτε να επιλέξετε το ένα(1) αρχείο του τύπου .mp4. Να σημειωθεί επίσης ότι το μέγιστο μέγεθος γιά το αρχείο πρέπει να είναι το πολύ 20ΜΒ.

    • Βίντεο Μετά: Πατώντας αυτό το το κουμπί μπορείτε να ανεβάσετε το βίντεο μετά την διακόσμηση για ένα συγκεκριμένο ακίνητο προκειμένου να μπορεί να προβληθεί στην ιστοσελίδα, αφού έχτε πρώτα επιλέξει τον αριθμό της διακόσμησης/ανακαίνισης για τον οποίο επιθυμήτε να ανεβάσετε αρχεία. Όταν πατήσετε το κουμπί choose files, μπορείτε να επιλέξετε το ένα(1) αρχείο του τύπου .mp4. Να σημειωθεί επίσης ότι το μέγιστο μέγεθος γιά το αρχείο πρέπει να είναι το πολύ 20ΜΒ.

    • Αντικατάσταση αρχείων: Σε περίπτωση που θέλετε να ανεβάσετε κανούργια αρχεία για το κάθε ακίνητο η διαδικασία είναι ακριβώς η ίδια με τα βήματα που αναφέρονται πιό πάνω. Να σημειωθεί ότι σε περιπτωσή αντικατάστασης των παλιών αρχείων με καινούργια, τα παλιά αρχεία θα διαγραφτούν απο το σύστημα. Αυτό σημαίνει ότι όταν θέλετε να αντικαταστήσετε για παράδειγμα μία μονο φωτογραφία, θα πρέπει να ξανα ανεβάσετε και τα παλιά σας αρχεία ξανά απο την αρχή.
