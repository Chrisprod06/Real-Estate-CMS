Interest List User Manual

On this screen you can view information and messages for clients interested in a particular property on the website. You can also print a report regarding the above information and delete messages that you no longer wish to see.

  1. Delete a customer from the list of interested parties:

    First click on the cart icon which is located in the last column in the stakeholders table and a window will open asking you if you want to delete the stakeholder. Then click delete and a window will open asking you if you want to ask if you want to delete the request. If you press yes then the customer's entry will be deleted from the system for that customer.

  2. Generate report:

    When you click on the Create Report button, a report will be created with the customer details and messages in PDF format.
