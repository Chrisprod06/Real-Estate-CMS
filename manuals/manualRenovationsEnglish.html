Εγχειρίδιο Χρήσης Διαχείριση Ανακαινίσεων

Σε αυτή την οθόνη μπορείτε να δείτε πληροφορίες για τις ανακαίνισεις που υπάρχουν στο συστήμα, ναπροσθέσετε λεπτομέρειες φωτογραφίες και βίντεο για νέες ανακαίνισεις καθώς και να τα διαγράψετε ή να επεξεργαστείτε τις λεπτομέρειες κάποιας ανακαίνισης. Επίσης μπορείτε να δημιουργήσετε αναφορά για τις ανακαίνισεις τις επιλογής σας.

  1. Εισαγωγή λεπτομερειών νέας ανακαίνισης:

    Αρχικά πατήστε το κουμπί Προσθήκη νέας ανακαίνισης και θα ανοίξει ένα παράθυρο . Στη συνέχεια εισάγετε όλα τα στοιχεία ( * σημαίνει ότι είναι υποχρεωτικό ) και πατήστε το κουμπί Προσθήκη.

  2. Εισαγωγή φωτογραφιών και βίντεο νέας ανακαίνισης:

    Αρχικά πατήστε το κουμπί Προσθήκη Νέας Ανακαίνισης. Στη συνέχεια διαλέξετε για ποιά ανακαίνιση θέλετε να εισάγετε φωτογραφίες και βίντεο ( * σημαίνει ότι είναι υποχρεωτικό ). Μπορείτε να εισάγετε φωτογραφίες για να εμφανίζονται στη προεπισκόπηση, στα καρουσέλ, αλλα και στο panorama view. Η φωτογραφία για τη προεπισκόπηση και για το καρουσέλ της αρχικής σελίδας πρέπει ΠΑΝΤΑ να είναι πρώτη. Μέτα μπορείτε να εισάγετε τις υπόλοιπες φωτογραφίες με όποια σειρά επιθυμήτε. και πατήστε το κουμπί Προσθήκη.

  3. Επεξεργασία λεπτομερειών υπάρχων ανακαίνισης:

    Αρχικά πατήστε το εικονίδιο του μολυβιού το οποίο βρίσκεται στο τελευταία στήλη στο πίνακα ανακαινίσεων και θα ανοίξει ένα παράθυρο με όλα τα υπάρχων στοιχεία για τη συγκεκριμένη ανακαίνιση. Όσα στοιχεία υποστούν αλλαγές θα ανανεωθούν στο σύστημα ενώ τα υπόλοιπα θα κρατήσουν της αρχικές τους τιμές.

  4. Επεξεργασία φωτογραφιών και βίντεο ανακαίνισης:

  5. Διαγραφή λεπτομερειών, φωτογραφιών και βίντεο ανακαίνισης:

    Αρχικά πατήστε το εικονίδιο του καλάθου το οποίο βρίσκεται στο τελευταία στήλη στο πίνακα ανακαινίσεων και θα ανοίξει ένα παράθυρο το οποίο σας ρωτά αν θέλετε να διαγράψετε την ανακαίνιση. Μετά πατήστε διαγραφή και θα διαγραφτούν όλα τα στοιχεία του ανακαίνισης καθώς και ότι φωτογραφίες και βίντεο υπάρχουν στο σύστημα για τη συγκεκριμένη ανακαίνιση.

  6. Δημιουργία αναφοράς:

    ταν πατήσετε το κουμπί Δημιουργία Αναφοράς θα δημιουργηθεί μια αναφορά με τα στοιχεία όλων των ανακαινίσεων σε μορφη pdf.

  7. Περιγραφές Πεδίων:

    • ID ακινήτου: Εδώ μπορείτε να διαλέξετε για ποιό ακίνητο θέλετε να προσθέσετε ανακαίνιση ή διακόσμηση. Πρέπει να είναι ακινητό με κατηγορία για ανακαίνιση ή διακόσμηση

    • Περιγραφή: Εδώ μπορείτε να πληκτρολογήσετε τη περιγραφή του ανακαίνισης. Το μέγιστο μέγεθος πρέπει να είναι 1200 χαρακτήρες.

    • Photos Before: By pressing this button you can upload photos before the decoration/renovation of the property in order to for it to be viewed in the carousel of the respective property on the website, after you have first selected the number of the property for which you wish to upload files. When you click on the choose files button, you can select a maximum of four (4) photos of type .png, .jpeg and .jpg by holding down the ctrl button, so that all of them get uploaded at the same time. Note also that the maximum size for each file must be a maximum of 20MB. The photo which will be uploaded first will be the photo that will appear in the thumbnail of the renovation grid on the website. Also photos must be selected in reverse order, depending on the order in which you want to upload them. For example, if the order you want to upload the files is (photo1, photo2, photo3) the selection from your computer should be (photo3, photo2, photo1).     • Photos After: By pressing this button you can upload photos after the decoration/renovation of the property in order to for it to be viewed in the carousel of the respective property on the website, after you have first selected the number of the property for which you wish to upload files. When you click on the choose files button, you can select a maximum of four (4) photos of type .png, .jpeg and .jpg by holding down the ctrl button, so that all of them get uploaded at the same time. Note also that the maximum size for each file must be a maximum of 20MB. The photo which will be uploaded first will be the photo that will appear in the thumbnail of the renovation grid on the website. Also photos must be selected in reverse order, depending on the order in which you want to upload them. For example, if the order you want to upload the files is (photo1, photo2, photo3) the selection from your computer should be (photo3, photo2, photo1).

    • Video Before: By pressing this button you can upload a video before the decoration/renovation for a specific property in order for it to be viewed on the website, after you have first selected the number of the decoration/renovation for which you wish to upload files. When you click on the choose files button, you can select one (1) file of type .mp4. Please also note that the maximum file size must be a maximum of 20MB.

    • Video After: By pressing this button you can upload a video after the decoration/renovation for a specific property in order for it to be viewed on the website, after you have first selected the number of the decoration/renovation for which you wish to upload files. When you click on the choose files button, you can select one (1) file of type .mp4. Please also note that the maximum file size must be a maximum of 20MB.

    • Update files: In case you want to upload new files for each property the procedure is exactly the same with the steps listed above. Please note that in case of replacing the old files with new ones, the old files will be deleted from the system. This means that when you want to replace, for example, a single photo, you will have to re-upload all your old files all over again.
