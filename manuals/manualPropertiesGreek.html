Εγχειρίδιο Χρήσης Διαχείριση Ακινήτων

Σε αυτή την οθόνη μπορείτε να δείτε πληροφορίες για τα ακίνητα που υπάρχουν στο συστήμα, ναπροσθέσετε λεπτομέρειες φωτογραφίες και βίντεο για νέα ακίνητα καθώς και να τα διαγράψετε ή να επεξεργαστείτε τις λεπτομέρειες κάποιου ακινήτου. Επίσης μπορείτε να δημιουργήσετε αναφορά για τα ακίνητα τις επιλογής σας.

  1. Εισαγωγή λεπτομερειών νέου ακινήτου:

    Αρχικά πατήστε το κουμπί Προσθήκη Νέου Ακινήτου και θα ανοίξει ένα παράθυρο . Στη συνέχεια εισάγετε όλα τα στοιχεία ( * σημαίνει ότι είναι υποχρεωτικό ) και πατήστε το κουμπί Προσθήκη.

  2. Εισαγωγή φωτογραφιών και βίντεο νέου ακινήτου:

    Αρχικά πατήστε το κουμπί Προσθήκη Νέου Ακινήτου. Στη συνέχεια διαλέξετε για ποιό ακίνητο θέλετε να εισάγετε φωτογραφίες και βίντεο ( * σημαίνει ότι είναι υποχρεωτικό ). Μπορείτε να εισάγετε φωτογραφίες για να εμφανίζονται στη προεπισκόπηση, στα καρουζέλ, αλλα και στο panoramic view. Η φωτογραφία για τη προεπισκόπηση και για το καρουζέλ της αρχικής σελίδας πρέπει ΠΆΝΤΑ να είναι πρώτη. Μέτα μπορείτε να εισάγετε τις υπόλοιπες φωτογραφίες με όποια σειρά επιθυμείτε. και πατήστε το κουμπί Προσθήκη.

  3. Επεξεργασία λεπτομερειών υπάρχων ακινήτου:

    Αρχικά πατήστε το εικονίδιο του μολυβιού το οποίο βρίσκεται στο τελευταία στήλη στο πίνακα ακινήτων και θα ανοίξει ένα παράθυρο με όλα τα υπάρχων στοιχεία για το συγκεκριμένο ακίνητο. Όσα στοιχεία στοιχεία υποστούν αλλαγές θα ανανεωθούν στο σύστημα ενώ τα υπόλοιπα θα κρατήσουν τις αρχικές τους τιμές.

  4. Επεξεργασία φωτογραφιών και βίντεο ακινήτου:

  5. Διαγραφή λεπτομερειών, φωτογραφιών και βίντεο ακινήτου:

    Αρχικά πατήστε το εικονίδιο του καλάθου το οποίο βρίσκεται στο τελευταία στήλη στο πίνακα ακινήτων και θα ανοίξει ένα παράθυρο το οποίο σας ρωτά αν θέλετε να διαγράψετε το ακίνητο. Μετά πατήστε διαγραφή και θα διαγραφούν όλα τα στοιχεία του ακινήτου καθώς και ότι φωτογραφίες και βίντεο υπάρχουν στο σύστημα για το συγκεκριμένο ακίνητο.

  6. Δημιουργία αναφοράς:

    Αρχικά πατήστε το κουμπί Δημιουργία Αναφοράς και θα εμφανιστεί ένα παράθυρο. Σε αυτο το παραθυρό επιλέξετε τις προτιμήσεις σας σχετικά με το για ποιά ακίνητα θέλετε να δημιουργήσετε αναφορά. Σε περίπτωση που δεν έχετε κάποια προτίμηση μπορείτε να μην επιλέξετε κάτι. Τέλος πατήστε το κουμπί Δημιουργία.

  7. Προδιαγραφές Πεδίων:

    • Τύπος: Εδώ μπορείτε να επιλέξετε αν το ακίνητο είναι σπίτι ή διαμέρισμα.

    • Κατηγορία:Εδώ μπορείτε να επιλέξετε αν το ακίνητο είναι για πούλημα, ενοικίαση βραχυπρόθεσμα ή μακροπρόθεσμα, ανακαίνιση ή διακόσμηση.

    • Χώρα: Εδώ μπορείτε να πληκτρολογήσετε σε ποιά χώρα βρίσκεται το ακίνητο.

    • Πόλη: Εδώ μπορείτε να πληκτρολογήσετε σε ποιά πόλη βρίσκεται το ακίνητο.

    • Περιφέρεια: Εδώ μπορείτε να πληκτρολογήσετε σε ποιά περιφέρεια βρίσκεται το ακίνητο.

    • Διεύθυνση: Εδώ μπορείτε να πληκτρολογείστε τη διεύθυνση του ακινήτου.

    • Υπνοδωμάτια: Εδώ μπορείτε να επιλέξετε πόσα υπνοδωμάτια έχει το ακίνητο.

    • Αποχωρητήρια: Εδώ μπορείτε να επιλέξετε πόσα αποχωρητήρια έχει το ακίνητο.

    • Επιπλα: Εδώ μπορείτε να επιλέξετε αν το ακίνητο έχει επιπλά ή όχι.

    • Χώρος Στάθμευσης: Εδώ μπορείτε να επιλέξετε αν το ακίνητο έχει χωρος σταθμευσης ή όχι.

    • Σύστημα Θέρμανσης: Εδώ μπορείτε να διαλέξετε αν το ακίνητο έχει σύστημα Θέρμανσης ή όχι.

    • Όροφος: Εδώ μπορείτε να επιλέξετε τον όροφο που βρίσκεται αν είναι διαμέρισμα ή τους ορόφους που έχει αν είναι σπίτι το ακίνητο.

    • Χρόνος Κατασκευής: Εδώ μπορείτε να πληκτρολογήσετε τη χρονολογία κατασκευής του ακινήτου. Πρέπει να είναι το μέγιστο 4 ψηφία.

    • Διαθέσιμο Από: Εδώ μπορείτε να επιλέξετε την ημερομηνία που θα είναι διαθέσιμο το ακίνητο.

    • Τετραγωνικά Μέτρα: Εδώ μπορείτε να πληκτρολογήσετε πόσα τετραγωνικά είναι το ακίνητα.

    • Συνολική Τιμή: Εδώ μπορείτε να πληκτρολογήσετε τη συνολική τιμή του ακινήτου.

    • Εμφάνιση στην αρχική: Εδώ μπορείτε να επιλέξετε αν θέλετε το ακίνητο να εμφανίζεται στο καρουζέλ της αρχικής σελίδας.

    • Τοποθεσία: Εδώ μπορείτε να εισάγετε το χάρτη της τοποθεσίας του ακινήτου. Για να δουλέψει σωστά αυτή η λειτουργία ΠΡΈΠΕΙ πρώτα να πάτε στη σελίδα https://www.google.com/maps/ . Έπειτα να επιλέξετε τη τοποθεσία του ακινήτου. Μετά πρέπει να πατήσετε στη όδο που εμφανίζει στο παραθυράκι στο κάτω μέρος της σελίδας. Μετά στη καρτέλα που θα εμφανιστεί πρέπει να πατήσετε το κουμπί share ή κοινοποίηση και μετά το embed a map. Τέλος βεβαιωθείτε πώς είναι επιλεγμένο το μέγεθος medium και πατήστε COPY HTML και κάντε το paste ή επικόλληση σε αυτό το πεδίο.

    • Περιγραφή: Εδώ μπορείτε να πληκτρολογήσετε τη περιγραφή του ακινήτου. Το μέγιστο μέγεθος πρέπει να είναι 1200 χαρακτήρες.

    • Ανέσεις: Εδώ μπορείτε να πληκτρολογήσετε τις ανέσεις που έχει το ακίνητο. Το μέγιστο μέγεθος πρέπει να είναι 1200 χαρακτήρες.

    • Φωτογραφίες: Πατώντας αυτό το το κουμπί μπορείτε να ανεβάσετε φωτογραφίες για ένα συγκεκριμένο ακίνητο προκειμένου να μπορούν να προβληθούν στο καρουζέλ του αντίστοιχου ακινήτου στην ιστοσελίδα, αφού έχει πρώτα επιλέξει τον αριθμό του ακινήτου για τον οποίο επιθυμείτε να ανεβάσετε αρχεία. Όταν πατήσετε το κουμπί choose files, μπορείτε να επιλέξετε το πολύ δέκα(10) φωτογραφίες του τύπου .png και .jpeg έχοντας πατημένο το κουμπί ctrl, ούτως ώστε να ανεβούν όλες οι φωτογραφίες ταυτόχρονα. Να σημειωθεί επίσης ότι το μέγιστο μέγεθος για κάθε αρχείο πρέπει να είναι το πολύ 20ΜΒ. Η φωτογραφία πού θα ανεβεί πρώτη θα είναι και η φωτογραφία που θα εμφανίζεται στο thumbnail του property grid στην ιστοσελίδα. Επίσης οι φωτογραφίες πρέπει να επιλεχθούν ανάποδα, ανάλογα με την σειρά που θέλετε να τις ανεβάσετε. Για παράδειγμα αν η σειρά που θέλετε να ανεβάσετε τα αρχεία είναι(photo 1, photo 2, photo 3) η επιλογή από τον υπολογιστή σας πρέπει να είναι(photo3, photo2, photo1).

    • Φωτογραφίες Πανόραμα: Πατώντας αυτό το το κουμπί μπορείτε να ανεβάσετε φωτογραφίες Πανόραμα για ένα συγκεκριμένο ακίνητο προκειμένου να μπορούν να προβληθούν στην γκαλερί του αντίστοιχου ακινήτου στην ιστοσελίδα, αφού έχετε πρώτα επιλέξει τον αριθμό του ακινήτου για τον οποίο επιθυμείτε να ανεβάσετε αρχεία. Όταν πατήσετε το κουμπί choose files, μπορείτε να επιλέξετε το πολύ δέκα(10) αρχεία του τύπου .jpg έχοντας πατημένο το κουμπί ctrl, ούτως ώστε να ανεβούν όλες οι φωτογραφίες ταυτόχρονα. Να σημειωθεί επίσης ότι το μέγιστο μέγεθος για κάθε αρχείο πρέπει να είναι το πολύ 20ΜΒ. Επίσης οι φωτογραφίες πρέπει να επιλεχθούν ανάποδα, ανάλογα με την σειρά που θέλετε να τις ανεβάσετε. Για παράδειγμα αν η σειρά που θέλετε να ανεβάσετε τα αρχεία είναι(photo 1, photo 2, photo 3) η επιλογή από τον υπολογιστή σας πρέπει να είναι(photo3, photo2, photo1).

    • Βίντεο: Πατώντας αυτό το το κουμπί μπορείτε να ανεβάσετε το βίντεο για ένα συγκεκριμένο ακίνητο προκειμένου να μπορεί να προβληθεί στην ιστοσελίδα, αφού έχετε πρώτα επιλέξει τον αριθμό του ακινήτου για τον οποίο επιθυμείτε να ανεβάσετε αρχεία. Όταν πατήσετε το κουμπί choose files, μπορείτε να επιλέξετε το ένα(1) αρχείο του τύπου .mp4. Να σημειωθεί επίσης ότι το μέγιστο μέγεθος γιά το αρχείο πρέπει να είναι το πολύ 20ΜΒ.

    • Αντικατάσταση αρχείων: Σε περίπτωση που θέλετε να ανεβάσετε καινούργια αρχεία για το κάθε ακίνητο η διαδικασία είναι ακριβώς η ίδια με τα βήματα που αναφέρονται πιό πάνω. Να σημειωθεί ότι σε περίπτωση αντικατάστασης των παλαιών αρχείων με καινούργια, τα παλιά αρχεία θα διαγραφουν απο το σύστημα. Αυτό σημαίνει ότι όταν θέλετε να αντικαταστήσετε για παράδειγμα μία μόνο φωτογραφία, θα πρέπει να ξανα ανεβάσετε και τα παλιά σας αρχεία ξανά από την αρχή.
