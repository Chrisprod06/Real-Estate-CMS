Renovation Management User Manual

In this screen you can view information about renovations that exist in the system, add details photos and videos of new renovations as well as delete them or edit the details of a renovation. You can also create a report on the renovations of your choice.

  1. Enter details of new renovation:

    First click the Add New Renovation button and a window will open . Then enter all the details ( * means it is mandatory ). and press the Add button.

  2. Insert photos and videos of new renovation:

    First click the Add New Renovation button. Click on the New Renovation button, then select which renovation to click on. you want to import photos and videos ( * means it is mandatory ). You can enter for which renovations you want to enter photos to be displayed in the preview, carousels, and panorama view. The photo for the preview and for the carousel of the home page must ALWAYS be first. After that you can insert the other photos in any order you wish. and press the Add button.

  3. Edit details of existing renovation:

    First click on the pencil icon which is located in the last column in the renovation table and a window will open with all the existing details for that particular renovation. All the items that will appear in the window with the new information will be displayed in the new window of the new renovation project. undergo changes will be updated in the system while the rest will keep their original values.

  5. Delete details, photos and renovation videos:

    First click on the reed icon which is located in the last column in the renovation panel and a window will open asking you if you want to delete the renovation. Then click delete and it will all details of the renovation will be deleted as well as any photos and videos that exist in the system for that particular renovation.

  6. Creating a report:

    When you click on the Create Report button, a report will be created with the details of all renovations in the format pdf.
