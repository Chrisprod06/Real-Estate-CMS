Property Management User Manual

On this screen you can view information about properties in the system, add details photos and videos of new properties as well as delete them or edit the details of a property. You can also create a report for the properties of your choice.

  1. Enter details of a new property:

    First click on the Add New Property button and a window will open . Then enter all the details ( * means it is mandatory ). If you click on the Add button.

  2. Insert photos and videos of new property:

    First click the Add New Property button. Click on the Add New Property button to open a property. you want to import photos and videos ( * means it is mandatory ). You can enter for which property you want to enter photos to be displayed in the preview, carousels, and panorama view. The photo for the preview and for the carousel of the home page must ALWAYS be the first one. After that you can insert the other photos in any order you wish. and press the Add button.

  3. Edit details of existing property:

    First click on the pencil icon which is located in the last column in the property table and a window will open with all the existing details for that property. All data items undergo changes will be updated in the system while the rest will keep their original values.

  5. Deleting details, photos and videos of the property:

    First click on the reed icon which is located in the last column in the property table and a window will open asking you if you want to delete the property. Then click delete and a window will open asking if you want to ask if you want to delete the property. all the details of the property will be deleted as well as any photos and videos that are in the system for that property.

  6. Creating a report:

    First click on the Create Report button and a window will appear. In this window select your preferences. on which properties you want to create a report for. In case you do not have a preference you can choose not to select something. Finally click the Create button.
