Manager's User Manual

In this screen you can view information about system administrators,add new administrators as well as delete them or edit the details of an administrator. You can also create a report for the administrators.

    .
  1. Introducing a new administrator:

    First click on the Add New Administrator button and a window will open . Then enter all the details ( * means it is mandatory ). If you click on the Add button.

  2. Edit existing administrator details:

    First click on the pencil icon which is located in the last column in the admin panel and a window will open with all the existing data for that administrator. All the data undergo changes will be updated in the system while the rest will keep their original values.

  3. Delete administrator:

    First click on the cart icon which is located in the last column in the admin panel and a window will open asking you if you want to delete the administrator. Then click delete and a window will appear asking you if you want to ask if you want to delete the cart. delete all the administrator data existing in the system for that administrator.

  4. Create report:

    When you click the Create Report button, a report will be created with the details of all administrators in the format pdf.
