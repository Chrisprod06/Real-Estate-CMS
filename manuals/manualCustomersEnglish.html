Customer Management User Manual

In this screen you can view information about the system's customers, delete or edit the details of a customer. You can also Create a report for customers.

    .
  1. Edit existing customer data:

    First click on the pencil icon which is located in the last column in the customer table and a window will open with all the existing data for that customer. All data undergo changes will be updated in the system while the rest will keep their original values.

  2. Customer deletion:

    First click on the cart icon which is located in the last column in the customer table and a window will open asking you if you want to delete the customer. Then click delete and a window will open asking you if you want to know if you want to delete the customer. delete all the customer details existing in the system for that customer.

  3. Generate report:

    When you click on the Create Report button, a report will be created with the details of all customers in the format pdf.
