Manage Newsletter User Manual

In this screen you can view information about the system customers who are subscribed to the newsletter , delete a customer from the newsletter and send newsletters to all customers subscribed to the newsletter . You can also create a report for customers who are subscribed to the newsletter .

    First click on the cart icon which is located in the last column in the newsletter customer table and a window will open asking you if you want to delete the customer. Then click unsubscribe and a window will open asking if you want to know if you want to unsubscribe. all the customer details existing in the system for that particular customer.

    When you click on the Create Report button, a report will be created with the details of all customers in the format pdf.
